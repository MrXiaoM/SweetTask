# SweetTask

Minecraft 每日任务插件

## 简介

想要给玩家找点事做吗？使用本插件来给玩家发布周期性的任务！每日、每周、每月三种任务周期，每种周期在不同权限组下自定义任务数量，任你选择。

任务目标（子任务）无论是破坏方块、放置方块、提交物品、合成物品、钓鱼、杀怪… 通通支持！

组合这些预设的任务目标制作任务配置，随机分发给玩家，完成后获得自定义命令奖励。

除此之外，你还可以为插件增加[更多菜单配置](https://plugins.mcio.dev/docs/task/menus)，就像一个菜单插件一样，任意修改界面布局。
